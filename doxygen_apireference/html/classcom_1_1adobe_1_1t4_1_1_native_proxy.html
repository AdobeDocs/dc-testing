void com.adobe.t4.NativeProxy.close ( )

Release the native memory held by this instance.

The method name is a little misleading. This method doesn't actually close anything. So, for example, in the derived class Document, Document.close() doesn't actually close the document. Rather, it releases the shared_ptr<document> held by the proxy. If this is the last reference to the document then it will in fact close. But if other native code still holds a shared_ptr to the same document, the document will not close.

Note that this method is called by finalize(). But callers should not count on finalize() being called, especially in a timely fashion. Clients should explicitly call close() when done with the instance.
