jbyteArray  buffer,
jint  byteOffset,
jint  byteCount

Native implementation of InputStream.read().

Attempts to read the requested number of bytes, and returns the number actually read. Returns 0 if and only if at end of file. Will block if necessary until some bytes are available or end of file is reached.

Unlike the corresponding Java method, this does not return -1 at end of file.
