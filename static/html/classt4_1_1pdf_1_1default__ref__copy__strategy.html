Detailed Description

Clients can provide their own strategy for reference copying (for example resolve all IDOs to direct objects) This default implementation copies each reference from the source to the destination only once, when the same source IDO is met again during copying, instead of creating new IDO in destination store the corresponding previously copied reference will be used
