Release Notes¶

August, 2021 Release¶

Welcome to the first release of the T4 SDK.

Available features¶

This milestone delivers:

  • Documentation

  • Support for building and developing on Windows, Linux, and Macintosh

  • Co-engineering infrastructure for T4 & ViewSDK

    • Joint library development documentation

    • CI/CD setup for Windows, Linux, and Macintosh

    • Library building support for Windows, Linux & Macintosh

  • Security features:

    • Boost: removed Regex and Range

    • Removed memory preamble in Allocators

    • Fuzzing: Addressed issues reported by Microsoft

    • Removal of Chrono, TypeTraits, MPL, Random, StaticAssert (originally M3)

    • Support for Partition Allocator

    • Removal and hardening of transient heap

  • PDF Viewing (originally M2)

    • Basic viewing

    • Pan

    • Zoom

    • Find

    • Copy

    • Layouts

    • Rotate view (without save)

System requirement changes¶

None (this is the initial release)
