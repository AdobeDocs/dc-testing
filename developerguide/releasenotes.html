Release Notes¶

August, 2021 Release¶

Welcome to the first release of the T4 SDK.

What’s new¶

Everything is new for this release, but this milestone delivers:

M1 features¶

  • Developer Guide and API Reference

  • Support for building and developing on Windows, Linux, and Macintosh

  • Co-engineering infrastructure for joint library development, including CI/CD setup for Windows, Linux, and Macintosh

  • Security features:

    • Boost: removed Regex and Range

    • Removed memory preamble in Allocators

    • Fuzzing: Addressed issues reported by Microsoft
