Getting Started with T4¶

T4 is a PDF library designed to keep even the most complex PDF files responsive and performant in resource-constrained environments. T4 gives memory consumption control to clients. Clients should expect T4 to successfully render content even with a very low threshold; performance improves as T4 is allowed to use more memory. T4 also requires all resources (chiefly memory) to be managed using the principle of RAII.
