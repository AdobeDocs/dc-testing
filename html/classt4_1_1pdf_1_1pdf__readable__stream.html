vector<t4::string> t4::pdf::pdf_readable_stream::get_filters ( ) const

Returns the value of the optional "Filter" property.

The returned vector contains filter names in the same order as they appear in the file. The vector will be empty if there is no Filter property, or if there is but its value is an array with 0 elements.
